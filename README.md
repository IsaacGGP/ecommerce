# ecommerce
Backend de ecommerce desarrollado con Spring Boot, implementando autenticación y autorización mediante JWT, manejo global de excepciones y pruebas automatizadas.

Tecnologías backend

- Java 21
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- PostgreSQL
- JPA / Hibernate
- JUnit 5
- Mockito
- MockMvc

Funcionalidades Implementadas (v1.0)

- Registro de usuarios
- Login con JWT
- CRUD de usuarios protegido
- Validaciones con Bean Validation
- Manejo global de excepciones
- Autenticación y autorización con Spring Security
- Cobertura de pruebas automatizadas

Testing

- Unit Testing con Mockito
- Controller Testing con MockMvc

Próximas funcionalidades (v2)

- Gestión de productos
- Categorías
- Carrito de compras
- Órdenes y pagos
- Inventario
- Roles avanzados (ADMIN / CUSTOMER)
